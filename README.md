## Descripción del Proyecto 📜




En este proyecto actual, hemos creado un carrito de compras especializado en productos de zapatillas 🛒. Para lograr este objetivo, puse en práctica los conocimientos adquiridos en el curso. Utilizando HTML, estructuré el contenido de manera efectiva 🧱, asegurándome de que la interfaz fuera clara y amigable para los usuarios 🤝. Además, implementé diseño web responsive 📱💻, que garantiza una experiencia óptima tanto en dispositivos móviles como en computadoras de escritorio.



Con CSS, apliqué estilos que no solo hicieron que la página fuera visualmente atractiva 🎨, sino que también aseguraron que la interfaz se adaptara de manera elegante a diferentes tamaños de pantalla 📏. Esto permitió que los usuarios disfrutaran de una experiencia de usuario consistente sin importar el dispositivo que estuvieran utilizando.


Además, implementé Javascript para agregar interactividad a la página 🕹️, aplicando lo aprendido en el curso de Platzi.

<img width="1493" alt="Captura de pantalla 2023-08-05 a la(s) 01 42 12" src="https://github.com/stypcanto/carritodecompras/assets/80213508/99db4d09-a540-41be-a8af-0aa008f3c2fc">

La funcionalidad clave que implementé es la capacidad de seleccionar un producto, lo cual desencadena un efecto visual en la interfaz. Cuando un usuario elige un producto, los fondos de los otros elementos se transforman en un suave tono de gris, resaltando así la selección. Cabe destacar que abordar este aspecto presentó ciertos desafíos durante mi proceso de aprendizaje.

Lograr que el fondo de los elementos adquiera ese tono gris mientras se excluye la barra de navegación de la transformación fue un paso especialmente complejo. Sin embargo, este obstáculo se convirtió en una oportunidad para profundizar en mis habilidades de desarrollo. 

<img width="1232" alt="Captura de pantalla 2023-08-05 a la(s) 01 42 26" src="https://github.com/stypcanto/carritodecompras/assets/80213508/ea2f9db5-ee09-4e37-8c75-f0af29a77b86">

Asimismo, implementé el diseño web responsive para brindar una experiencia visualmente atractiva al buscar productos. A través de estilos CSS cautivadores, logré que la búsqueda de productos se adapte de manera fluida a distintos tamaños de pantalla. Este enfoque garantiza que los usuarios puedan explorar la búsqueda de productos de manera cómoda y atractiva, sin importar el dispositivo que utilicen.

<img width="632" alt="Captura de pantalla 2023-08-05 a la(s) 01 42 37" src="https://github.com/stypcanto/carritodecompras/assets/80213508/ec1b8825-2d12-4390-a990-d01ad640ab85">

Además, he habilitado la visualización del carrito de compras directamente desde el menú de diseño web responsive. Esto significa que, independientemente del dispositivo que estés usando, podrás acceder fácilmente al carrito de compras





<img width="318" alt="Captura de pantalla 2023-08-05 a la(s) 01 42 51" src="https://github.com/stypcanto/carritodecompras/assets/80213508/67a0eb2d-f52a-4703-97b8-f49bcdfc7ce9">

En situaciones en las que un producto no se encuentra disponible, la página ahora muestra una elegante pantalla de error 404. He diseñado estilos CSS sumamente atractivos y dinámicos para esta página de error. De esta manera, incluso en momentos de búsqueda infructuosa, los usuarios son recibidos con un diseño visualmente agradable que refleja la atención al detalle que he puesto en todo el proyecto
