# Yard Sale

Es una tienda online sobre venta de productos de segunda mano. La página cuenta con un diseño responsivo, conectado a una API Store de Platzi.

#### Skills Tecnicos.
- CSS.
- HTML.
- Manejo del DOM.
- Consumo de API.
- Node.
- NPM (manejador de packetes de Node.js).
- GitHub.
- Mobile First.
